Quick Translator

Quick Translator가 성공적으로 설치되었습니다! 이제 간편하게 텍스트를 번역해보세요.

간편한 번역

웹페이지에서 텍스트를 선택하기만 하면 자동으로 번역 결과가 툴팁으로 나타납니다.

양방향 번역

선택한 텍스트가 기본 언어(예: 한국어)면 학습 언어(예: 영어)로, 외국어라면 기본 언어로 자동 번역됩니다.

맞춤 설정

사이드 패널에서 기본 언어, 학습 언어, API 설정 등을 자유롭게 변경할 수 있습니다.

사용 방법

1
브라우저 툴바에서 Quick Translator 아이콘을 클릭하거나 Ctrl+B (Mac에서는 Cmd+B)를 눌러 사이드 패널을 열어보세요.
2
웹페이지에서 번역하고 싶은 텍스트를 선택하면 자동으로 툴팁이 나타납니다.
3
사이드 패널의 설정 탭에서 기본 언어와 학습 언어를 변경할 수 있습니다.
